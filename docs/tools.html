Tools

Illuminator provides six interactive tools, accessible via the tool tray or number keys 1–6. Hold Shift at any time to temporarily switch to the Translation tool, and release to return to the previous tool.

Translation Tool [1]

The default tool. Click and drag to move selected nodes. This is the primary tool for manually positioning nodes in your graph.

  • Left-click + drag to move selected nodes

Rotation Tool [2]

Rotate the current selection around its center. Click and drag to apply rotation interactively.

  • Left-click + drag to rotate selected nodes

Scaling Tool [3]

Scale the current selection relative to its center. Drag to uniformly scale the positions of selected nodes closer together or farther apart.

  • Left-click + drag to scale the selection
  • Hold Alt for finer control
  • Press Esc to cancel

Box Selector [4]

Click and drag to draw a rectangular selection box. All nodes inside the box will be selected when the mouse is released.

  • Left-click + drag to draw a selection rectangle
  • A small click toggles the node under the cursor
  • Hold Ctrl to add to the existing selection instead of replacing it

Line Layout Tool [5]

Arranges the selected nodes in a straight line, distributing them evenly.

Arc Layout Tool [6]

Arranges the selected nodes along a circular arc. Drag to interactively adjust the arc's angle and radius.

  • Left-click + drag to adjust the arc shape
